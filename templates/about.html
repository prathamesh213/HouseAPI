Project Overview

This House Price Prediction application uses machine learning to estimate property values based on key features such as the number of bedrooms, bathrooms, floors, and the year the house was built. The model is trained on historical housing data to provide accurate price predictions.

Technology Stack

Backend

Python with Flask framework

Machine Learning

Scikit-learn with Linear Regression

Frontend

HTML, CSS, Bootstrap 5

Data Processing

Pandas, NumPy
